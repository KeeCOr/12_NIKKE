- 대표 피드백 축: 화력 부족, 생존 라인 붕괴, 약점 집중 성공/실패, 재장전 겹침을 분리해 설명한다.
- 검증: Unity Editor 라이선스가 없어 EditMode 테스트/빌드는 실행되지 않았고, `_temp/logic-check` .NET 검증으로 CombatAdvisorLogic 결과 분기 4건을 확인했다.
