최신 변경 사항

미니언 타겟팅

  • 일반 미니언은 더 이상 배열상 첫 번째 바리케이드만 향하지 않는다.
  • 현재 위치에서 가장 가까운 살아있는 바리케이드를 선택한다.
  • 여러 경로에서 스폰된 적이 하나의 바리케이드에 과도하게 몰려 이동이 멈춘 것처럼 보이는 현상을 줄인다.
  • 바리케이드 파괴 시 해당 렌더러를 비활성화해 다음 타겟 선택이 명확하게 넘어가도록 한다.

바리케이드 배치

  • 바리케이드 3개의 좌표를 더 넓게 분산했다.
  • Barricade0: (3.2, 1.34)
  • Barricade1: (5.0, 2.10)
  • Barricade2: (6.8, 2.86)

조준 표시

  • 선택 캐릭터가 빈 공간을 드래그해 조준하면 free-aim 타겟으로 유지된다.
  • 사정거리 밖이라도 조준선과 조준점은 계속 표시된다.
  • 발사 가능 여부는 기존처럼 무기 사정거리로 제한하며, 사거리 밖 조준점은 시각 피드백만 제공한다.

검증

  • EditMode 테스트에 바리케이드 근접 타겟 선택 테스트를 추가했다.
  • EditMode 테스트에 free-aim 조준점 유지 테스트를 추가했다.
