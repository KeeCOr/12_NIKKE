## 1. 게임 컨셉

### 1.1 핵심 비전

> "5명의 스쿼드가 거대 보스와 잡몹 무리가 쏟아지는 전장에서, 장벽과 바리케이드를 사수하며 부위파괴로 보스를 무력화시키는 전략 방어 게임"

### 1.2 뷰 구성 (45도 쿼터뷰 — 뒷모습 + 사이드 중간)

카메라가 캐릭터 후방 45도 위치에서 바라보는 구성. 도심 배경이 원근감 있게 펼쳐지며 보스가 건물 사이를 부수며 접근.

```
┌───────────────────────────────────────────────────┐
│    [하늘] [도심 빌딩 실루엣 원거리]                │
│   /건물\  /건물\    [작은 몬스터 무리]  /건물\    │
│  │      │         ↓ 접근 중...        │      │   │
│  │      │   [보스 — 건물 사이로 출현]  │      │   │
│  │  🏢  │     HEAD                   │  🏢  │   │
│  │  (파│  ARM_L CHEST★ ARM_R         │  (온│   │
│  │  손)│     LEG_L LEG_R             │  전)│   │
│  ╲____╱ ─────────────────────────── ╲____╱   │
│   [건물 잔해/연기] [장벽 HP████████]            │
│   ─── [바리케이드] ────────────────────────── │
│  [A 45°][B 45°][C 45°][D 45°][E 45°]          │
│    알파   브라보  찰리   델타   에코             │
│   [HP██] [HP██] [HP██] [HP██] [HP██]           │
└───────────────────────────────────────────────────┘
 ↑ 약 45도 쿼터뷰 — 캐릭터 옆면 30%+뒷면 70% 노출
   보스가 가까워질수록 화면상 크기 증가 (스케일 효과)
```

### 1.3 게임 흐름 요약
1. 보스 + 잡몹이 원거리에서 접근 시작
2. 잡몹은 선봉대로 먼저 달려와 **바리케이드** 파괴 시도 or 스쿼드 공격
3. 보스는 천천히 접근하며 **장벽** 파괴
4. 스쿼드는 자동 사격 중 — 플레이어가 집중 조준 지시
5. 보스 HP 50% + 특정 부위 파괴 시 **광폭화** 전환
6. 보스 HP 0 → **승리** / 보스가 방어선 돌파 or 스쿼드 전멸 → **패배**

## 3. 보스 시스템 상세

### 3.1 부위 시스템

| 부위 ID | 이름 | HP | 데미지 배율 | 파괴 효과 | 광폭화 조건 기여 |
|---------|------|----|-------------|-----------|----------------|
| HEAD | 머리 | 250 | ×2.0 | 공격 패턴 일부 제거 | ✅ |
| ARM_L | 왼팔 | 150 | ×1.0 | 공격력 40% 감소 | — |
| ARM_R | 오른팔 | 150 | ×1.0 | 공격력 40% 감소 | — |
| LEG_L | 왼다리 | 180 | ×1.0 | 이동속도 25% 감소 | — |
| LEG_R | 오른다리 | 180 | ×1.0 | 이동속도 25% 감소 | — |
| CHEST | 흉부 | 350 | ×1.5 | 잡몹 소환 쿨다운 증가 | ✅ |
| CORE ★ | 핵심 약점 | 120 | ×3.0 | 즉시 300 대미지 + 스턴 | 노출 조건 필요 |

### 3.2 핵심 약점(CORE) 노출 조건
- **CHEST 파괴** 시 CORE 히트박스 활성화
- 기본 상태에서 CORE는 타격 불가 (흉부로 가려져 있음)

### 3.3 광폭화 조건
```
광폭화 트리거 = (보스 HP ≤ 50%) AND (HEAD 또는 CHEST 파괴됨)
```
- 둘 중 하나라도 미충족 시 광폭화 없음
- 광폭화 발동 시:
  - 눈 색상 → 빨강, 전신 붉은 오라 이펙트
  - 이동속도 +50%
  - 공격력 +80%
  - 잡몹 소환 주기 2배 빨라짐
  - 지형지물 파괴 속도 2배
  - **새로운 패턴**: 전방 충격파 공격 추가

### 3.4 보스 행동 패턴

| 상태 | 패턴 | 설명 |
|------|------|------|
| 일반 | 전진 | 일정 속도로 접근. 장벽 앞에서 정지 후 파괴 |
| 일반 | 잡몹 소환 | 주기적으로 작은 몬스터들 소환 |
| 일반 | 포효 공격 | 주기적으로 충격파 발사 (스쿼드 or 지형 피해) |
| 광폭화 | 돌진 | HP 25% 이하 시 가끔 돌진하며 바리케이드 즉시 파괴 |
| 광폭화 | 충격파 | 넓은 범위의 충격파로 스쿼드 전체 피해 |

## 4. 잡몹 시스템

### 4.1 잡몹 종류

| 종류 | 이름 | 역할 | 행동 |
|------|------|------|------|
| 러너 | Runner | 선봉대 | 빠르게 달려와 바리케이드 파괴 or 스쿼드 근접 공격 |
| 버서커 | Berserker | 중형 | 느리지만 강함. 장벽에 데미지 줄 수 있음 |
| 스피터 | Spitter | 원거리 | 멀리서 산성 투사체 발사. 스쿼드 공격 |

### 4.2 잡몹 행동 우선순위
```
1. 바리케이드가 살아있으면 → 바리케이드 공격
2. 바리케이드 파괴 후 → 스쿼드 직접 공격
3. 스피터는 항상 원거리에서 스쿼드 공격
```

## 5. 지형지물 시스템

### 5.1 지형지물 구성

```
[보스/잡몹 접근 방향]  ← 도심 도로 위를 걸어옴
         ↓
┌────────────────────────────────────────────┐
│  🏢 [도심 건물 A — HP: 800]  🏢 [건물 B]   │ ← 가장 먼저 보스가 통과
│     (보스가 부수며 지나감 — 잔해 낙하)      │
│                                            │
│  [장벽 Wall — HP: 1000]                    │ ← 보스가 정면 파괴
│  (건물 잔해가 장벽에 쌓여 HP 일부 손상)     │
│                                            │
│  [바리케이드 — HP: 300] [바리케이드 — HP: 300]│ ← 잡몹(러너)이 파괴
│                                            │
│  [스쿼드 포지션]                            │
└────────────────────────────────────────────┘
```

### 5.2 지형지물 상태 효과

| 지형지물 | 파괴자 | HP | 파괴 시 효과 |
|---------|--------|----|-------------|
| 도심 건물 | 보스 (통과) | 800 | 낙하 잔해 → 잡몹 일부 피해 + 장벽에 10% 추가 피해. 연기/먼지 이펙트 |
| 장벽 | 보스 + 버서커 | 1000 | 스쿼드 방어 보너스 제거. 보스 직접 공격 범위 진입 |
| 바리케이드 | 잡몹(러너) | 300 | 러너들이 스쿼드 직접 공격 가능 |

### 5.3 장벽 HP 상태 표시
- HP > 70%: 정상 (회색/파란색)
- HP 30-70%: 균열 (노란색, 균열 텍스처)
- HP < 30%: 심각 (빨간색, 연기 이펙트)
- HP 0: 붕괴 애니메이션 → 잔해만 남음

## 6. 스쿼드 시스템

### 6.1 캐릭터 구성

| 캐릭터 | 무기 | 탄창 | 공격력 | 사격속도 | HP | 특기 |
|--------|------|------|--------|----------|-----|------|
| Alpha | 볼트액션 저격소총 | 5발 | 120 | 2.5s | 80 | 약점 데미지 +50% |
| Bravo | 돌격소총 (AR) | 30발 | 30 | 0.15s | 100 | 연사 시 집탄율 증가 |
| Charlie | 산탄총 (Shotgun) | 8발 | 15×6 | 1.2s | 120 | 근거리 시 범위 피해 |
| Delta | 로켓런처 | 2발 | 200 | 4.0s | 70 | 폭발 범위 피해 (잡몹 군집 효과적) |
| Echo | 저격 저격소총 | 10발 | 60 | 1.0s | 90 | 적 약점 마킹 (아군 +20% 데미지) |

### 6.2 조준 방식
- **기본**: 각 캐릭터가 현재 조준 중인 부위에 자동 사격
- **집중 조준**: 보스 부위를 클릭 또는 드래그 → 가장 가까운 캐릭터 or 선택된 캐릭터가 해당 부위 조준
- **잡몹 대응**: 잡몹이 근접 시 자동으로 가장 가까운 잡몹에 조준 전환

### 6.3 탄약 및 장전

```
탄창 UI:  [●●●●●●●●●●○○○○○○○○○○]
                ↑현재 탄약     ↑비어있음

자동 장전 발동: 탄창 = 0
장전 시간: 캐릭터별 다름 (Alpha: 3s, Bravo: 2s, Charlie: 1.5s, Delta: 4s, Echo: 2s)
장전 중: 해당 캐릭터 사격 불가 + 장전 애니메이션 재생
장전 완료: 자동으로 사격 재개
```

### 6.4 스쿼드 HP 및 사망
- 잡몹(러너/스피터)에게 맞으면 개인 HP 감소
- HP 0 → 해당 캐릭터 쓰러짐 애니메이션 → 비활성화
- 전원 사망 → 게임 오버

## 7. 기술 스택 및 아키텍처

### 7.1 기술 스택

```
Phaser.js 3 (v3.60+)
├── Scene System (Boot, Preload, Menu, Game, GameOver, Win)
├── Physics: Arcade Physics (간단한 충돌 검사)
├── Tweens + Particles (이펙트)
├── Camera (고정 카메라, UI 레이어 분리)
└── Input: Pointer/Touch (드래그 조준)

Build: 없음 (단일 HTML + CDN Phaser)
```

### 7.2 파일 구조

```
/
├── index.html
├── assets/
│   ├── sprites/
│   │   ├── boss/          # 보스 스프라이트시트 (각 부위별 상태)
│   │   ├── squad/         # 5명 캐릭터 뒷모습 스프라이트
│   │   ├── minions/       # 잡몹 3종 스프라이트
│   │   └── terrain/       # 장벽, 바리케이드 상태별
│   ├── effects/           # 파티클, 폭발 이펙트
│   └── audio/             # 총성, 피격음, 보스 포효
└── js/
    ├── main.js              # Phaser 게임 초기화
    ├── scenes/
    │   ├── BootScene.js     # 에셋 로드
    │   ├── GameScene.js     # 메인 게임 로직
    │   ├── UIScene.js       # HP바, 탄약, 쿨다운 (별도 씬)
    │   └── ResultScene.js   # 승패 화면
    ├── entities/
    │   ├── Boss.js          # 보스 클래스 (이동, 패턴, 부위)
    │   ├── BossPart.js      # 부위별 히트박스 + HP
    │   ├── Minion.js        # 잡몹 기본 클래스
    │   ├── SquadMember.js   # 스쿼드 캐릭터 기본 클래스
    │   └── Bullet.js        # 투사체
    ├── systems/
    │   ├── AimSystem.js     # 조준 + 드래그 입력 처리
    │   ├── FireSystem.js    # 자동사격 + 장전 관리
    │   ├── TerrainSystem.js # 지형지물 상태 관리
    │   ├── WaveSystem.js    # 잡몹 소환 웨이브
    │   └── EnrageSystem.js  # 광폭화 조건 체크 + 발동
    └── config/
        ├── BossConfig.js    # 보스 스탯, 부위 정의
        ├── SquadConfig.js   # 5 캐릭터 스탯
        └── MinionConfig.js  # 잡몹 3종 스탯
```

### 7.3 씬 레이어 구조

```
UIScene (top)     : HP바, 탄약 UI, 광폭화 경고, 게임 상태
GameScene         : Background → Terrain → Minions → Boss → Squad → Effects
```

## 8. 게임 상태 머신

```
BOOT → PRELOAD → MENU
                  ↓
               PLAYING ─── PAUSED
                  ↓ ↘
                WIN   LOSE
                (보스 처치)  (전멸 or 방어선 붕괴)

PLAYING 내부:
  NORMAL_PHASE ──[조건 충족]──→ ENRAGE_PHASE
                                ↓
                           LAST_STAND (HP < 25%)
```
