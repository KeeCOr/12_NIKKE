게임 개요

SquadVsMonster는 5명의 분대원을 지휘하여 거대 보스 몬스터의 진격을 막는 실시간 전략 디펜스 게임입니다. 플레이어는 분대원을 선택·조준하고, 보스의 각 신체 부위를 정밀 타격해 효율적으로 격파해야 합니다.

보스 처치 후에는 로그라이크 강화 루프를 통해 분대를 업그레이드하고 점점 강해지는 다음 스테이지에 도전합니다.

핵심 차별점 — "부위 파괴"에 따른 보스 행동 변화 + 선택적 조준으로 인한 즉각적인 전략적 피드백

게임 목표

보스 HP 0 → 승리 방벽 HP 0 → 보스 진격 → 방어선 돌파 → 패배 분대 전멸 → 패배

디버프 시스템

부위를 타격하면 5초 지속 디버프가 적용됩니다. 이미 파괴된 부위를 타격하면 강화 디버프(DebuffStrong).

대상 부위디버프 효과강화 시
HEAD / CORE받는 피해 +15%+30%
LEG_L / LEG_R이동속도 ×0.95×0.90
ARM_L / ARM_R공격간격 ×1.20×1.40
CHEST스턴 확률 8%20% (스턴 1.5s)

방어선 (Defense Line)

X = 1.3 — 보스가 이 선을 넘으면 즉시 패배. 분대원은 이 선 앞(X≥0.68)에 배치.

입력 & 조작

Mouse / Touch 입력을 지원합니다. New Input System과 Legacy Input System 모두 대응.

조작행동
분대원 클릭선택 / 재클릭 시 선택 해제. 선택된 멤버 하이라이트
선택 후 드래그조준 방향 지정 (DragTarget). 보스 부위 / 미니언 위에서 떼면 타겟 고정
빈 영역 클릭드래그 시작 (비선택 상태)
BOMB 버튼 클릭폭격 활성화 (쿨타임 완료 시만 활성)

선택 하이라이트

선택된 분대원의 bodyRenderer 색상이 Lerp(기본색, 흰색, 0.45)로 밝아집니다.
